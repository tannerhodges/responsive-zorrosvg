  • From the DevTools Timeline, the image is decoded and rendered once the first byte is returned. Nothing tells Chrome to repaint after the image is fully downloaded, so you get the pixelated first byte version. Note: the full quality image is repainted once you resize the page.
  • The first byte image is most noticeable when referencing an external image. If you load a local image immediately, the problem is resolved.
